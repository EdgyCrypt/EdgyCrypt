## Using Virtual Env

A virtual environment will allow for us to keep the python version and packaging information separate from the python that sits on your system 

### Activating your Virtual Env

Windows:
```bash
env\bin\activate
```

MacOS and Linux:
```bash
source env/bin/activate
```
